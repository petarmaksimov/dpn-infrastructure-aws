```bash

```

## RDS PostgreSQL credentials

The DEV RDS PostgreSQL administrator password is generated by OpenTofu using the `random_password` provider and stored in AWS Secrets Manager.

Secret name:

```text
dpn/dev/postgres/admin
```

The secret JSON contains:

```json
{
  "username": "dpnadmin",
  "password": "<generated>",
  "engine": "postgres",
  "host": "<rds-endpoint>",
  "port": 5432,
  "dbname": "dpn"
}
```

No database password is stored in `dev.tfvars`, GitHub Actions, or the repository.
